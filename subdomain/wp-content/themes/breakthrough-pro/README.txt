BREAKTHROUGH PRO
https://my.studiopress.com/themes/breakthrough/

INSTALL
1. Upload the Breakthrough Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Breakthrough Pro theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking. To set up the theme like the demo, please visit https://my.studiopress.com/documentation/breakthrough-pro-theme/.

CHANGELOG

= 1.3.0 =

Breakthrough Pro 1.3.0 requires WordPress 5.4 or higher.

* Added: CSS styles for WordPress 5.4 Social Icons.
* Added: Editor styles to ensure theme colors work for inline text colors.
* Added: Automatic color contrast for theme background color palette.
* Changed: Link styling for better accessibility.
* Changed: Combined all Block Editor and theme appearance options into a single file at config/appearance.php.
* Changed: WooCommerce block CSS styling to be consistent with other WooCommerce styles.
* Changed: Blockquote styles for better integration with the block editor.
* Changed: Editor and front-end styles for WordPress 5.4 and Gutenberg plugin 7.9+ compatibility, including link color, buttons, headings, and lists.
* Changed: Remove rem units.
* Changed: Use genesis_get_theme_handle() and genesis_get_theme_version() functions from Genesis 3.0.0 in place of declaring CHILD_THEME_NAME and CHILD_THEME_VERSION constants.
* Changed: Use Google Fonts font-display &display=swap query parameter.
* Fixed: Comment reply link color when a custom color is set in the customizer.
* Fixed: Genesis eNews Extended button position.
* Fixed: Prevent an issue where the WooCommerce shop page layout could be ignored and fall back to the site default layout on sites that use object caching.
* Fixed: Ensure that general list styles do not apply to WooCommerce product blocks.
* Fixed: Full-width alignment Cover block width.

= 1.2.0 =
* Added: Theme support for HTML5 script and style elements (requires WordPress 5.3+).
* Added: Genesis lazy load image support (requires Genesis 3.2+).
* Added: WordPress 5.3 alignment classes.
* Changed: Separator and hr styles to allow for block color options.
* Changed: CSS for image, gallery, and gallery item figcaptions.
* Changed: CSS for tables for better consistency between editor and front end and to allow for new WordPress 5.3 settings.
* Changed: Moved theme supports to single `config/theme-supports.php` file.
* Changed: Moved theme defaults to config file at `config/child-theme-settings.php`.
* Changed: Improvements to CSS and PHP coding standards.
* Removed: Navigation extras filters since they are no longer necessary in Genesis 3.0+.
* Removed: Genesis settings page filters since they are no longer necessary in Genesis 3.0+.
* Removed: Squared button styles in favor of new button block border-radius setting.
* Removed: viewport meta tag code from functions.php. This is now added by Genesis.
