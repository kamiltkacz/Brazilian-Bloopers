NAVIGATION PRO
https://my.studiopress.com/themes/navigation/

INSTALL
1. Upload the Navigation Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the theme & use Genesis one-click theme setup to create your new homepage.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking. For additional instructions, please visit https://my.studiopress.com/documentation/navigation-pro-theme/.

Visit Genesis > Child Theme Setup to restart the one-click theme setup, if desired.
